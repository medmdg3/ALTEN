Construct a polynomial shifted to the right or to the left by a certain value \(|n|\),
it erases the first \(|n|\) elements if it is being shifted to the right, or multiplying the polynom by \(X^{|n|}\) if the shift is to the left, while filling the new coeficiant with a given value.

Parameters
AThe polynom containing the original values
nDefines how the polynomial will be shifted:
It will be shifted to the left by \(|n|\) if \(n>=0\)
It will be shifted to the right by \(|n|\) if \(n<0\)
valThe given value
Returns
The constructed polynomial
