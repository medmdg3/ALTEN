This function solves the finite horizon Discrete Linear Quadratic Regulator (DLQR) problem. The underlying equation is $ x_{t+1} = Ax_t + Bu_t $. The goal is to find values of $ u $ that minimize $ \sum_{t=0}^{N-1} (x_t^T Q x_t + u_t^T R u_t) + x_N^T Q_f x_N $, where $ Q = Q^T \geqslant 0 $, $ Q_f \geqslant 0 $, $ R = R^T > 0 $, and $ A $, $ B $, $ Q $, $ R $, $ Q_f $, $ x_0 $, and $ N $ are fixed. ~\newline
The main source of the algorithm idea\+: \href{https://stanford.edu/class/ee363/lectures/dlqr.pdf}{\texttt{ https\+://stanford.\+edu/class/ee363/lectures/dlqr.\+pdf}}.
